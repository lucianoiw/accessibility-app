texto-justificado LIKELY

Violacao Texto com alinhamento justificado:

Este paragrafo usa text-align: justify, o que pode criar espacos irregulares entre palavras e dificultar a leitura para pessoas com dislexia ou baixa visao. A WCAG 1.4.8 (AAA) recomenda evitar texto justificado para melhorar a legibilidade. O problema e especialmente grave em colunas estreitas onde os espacos ficam muito grandes.

Como testar:
1. Sirva este arquivo: npx serve acctest
2. Crie um projeto apontando para a URL (ex: http://localhost:3000)
3. Inicie uma auditoria com COGA habilitado
4. Verifique os badges de confianca nas violacoes detectadas

Filtros disponiveis na UI:
- Filtrar por nivel de confianca (Certain, Likely, Needs Review)
- Mostrar/ocultar regras experimentais
- Badges visuais com tooltip mostrando sinais de confianca

Testar IGT (Intelligent Guided Tests):
1. Apos a auditoria, clique em "Iniciar Testes" no card de Testes Guiados
2. Voce devera ver 2 IGTs disponiveis:
  - Qualidade do Alt Text (image-alt, imagem-alt-nome-arquivo)
  - Proposito dos Links (link-texto-generico, link-name, rotulo-curto-ambiguo)
3. Clique em "Iniciar teste" para responder as perguntas guiadas
4. O IGT de Libras so aparece para sites .gov.br
